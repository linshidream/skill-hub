# Generic Browser Adapter

Use this when the agent has no built-in browser automation.

1. Open the Mafengwo URL in a normal browser.
2. Complete safety verification manually if shown.
3. Open DevTools Console.
4. Paste the browser extraction snippet from `SKILL.md`.
5. Save the copied output as `raw_image_links.txt`.
6. Run `scripts/download_original_images.py`.
