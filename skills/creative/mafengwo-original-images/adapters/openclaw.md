# OpenClaw Adapter

If the OpenClaw CLI is available:

```bash
openclaw skills install ./skills/mafengwo-original-images --as mafengwo-original-images
```

Or use the hub installer:

```bash
./scripts/install.sh mafengwo-original-images --agent openclaw
```

For a hub repository that contains many skills, prefer installing a specific skill directory rather than the repository root.

Prompt example:

```text
Use mafengwo-original-images to download original images from:
https://www.mafengwo.cn/photo/10051/scenery_24796451_1.html
```
